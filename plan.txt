我要开发一个聊天工具，前端使用vue3和其他热门的css框架，后端使用fastapi。要实现打开页面可以随机一个用户名，可以一键创建聊天室，也可以在聊天室列表中查看要进去哪个id，或者直接输入聊天室id也可以。创建聊天室可以选择是否使用密码加密。聊天功能要能够发送 文字 表情 图片
  和视频。

✦ 这是一个非常棒的全栈项目想法。为了构建这个聊天工具，我建议采取以下开发计划。

  开发计划提案

  1. 技术栈选择
   * 前端: Vue 3 (使用 Vite 构建) + Tailwind CSS (现代、灵活的 CSS 框架) + Pinia (状态管理) + Vue Router。
   * 后端: FastAPI (Python) + WebSockets (实现实时聊天) + SQLite (轻量级数据库，用于存储房间和聊天记录)。
   * 文件存储: 本地文件系统 (用于存储上传的图片和视频)。

  2. 核心功能实现方案
   * 用户系统: 前端生成随机用户名 (例如: "匿名用户-123") 并存储在本地 (localStorage)，无需复杂的注册登录。
   * 聊天室管理:
       * 创建: API 接口支持设置房间名和可选密码。
       * 列表: 实时获取当前活跃的房间列表。
       * 加入: 校验密码（如果有），通过 WebSocket 连接特定房间频道。
   * 消息功能:
       * 文本/表情: 使用标准 WebSocket 传输文本。
       * 图片/视频: 先通过 HTTP POST 上传到后端，后端返回文件 URL，再通过 WebSocket 发送包含 URL 的消息，前端渲染为 <img> 或 <video>。

  3. 项目结构
  我们将创建两个主要文件夹：
   * /frontend: Vue 3 项目代码。
   * /backend: FastAPI 项目代码。
